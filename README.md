## Project Title
Hospital Patient Management System

## Project Description
The Hospital Patient Management System is a command-line application designed to simplify hospital record handling. It allows staff to add, update, and search patient records, manage waiting and admitted patient queues, and store data safely in files. The system is easy to use, reduces paperwork, saves time, and minimizes human error.
